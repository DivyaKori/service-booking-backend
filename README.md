# Service Booking Backend

A Flask-based backend for service booking with:
- Slot management
- No double booking
- Booking 
- Firebase Firestore integration

## Tech Stack
- Python (Flask)
- Firebase Firestore
- REST APIs

## Features
- Services: Haircut, Spa & Salon, Room Booking
- Slot-based booking system
- Safe backend validation

## How to Run
1. Clone the repo
2. Add Firebase service key (`firebase_key.json`)
3. Run:
   ```bash
   python app.py
